This site will include instructions on how to create your Github portfolio as they are developed

  1. Go to documents Folder and create a folder called "[YourName]PortfolioSite"
  2. Download this zip file here
    3. Sample Student Portfolio
  3. There will be folder called SamplestudentPortfolio with 5 HTML files inside of it. When you double click this .zip file, it should automatically expand, creating a folder with 5 HTML files inside of it (index.html, WCP.html, etc.)
  4. "Drag And Drop" or "Copy and Paste" Those 5 HTML files (not the folder) into your new Site Folder on the computer)
  5. Open Dreamweaver and Go to Site (at the Top Toolbar) --> New Site.
    For Site Name write: [Your Name] Portfolio.
    For the local site folder, choose the folder that you created.
