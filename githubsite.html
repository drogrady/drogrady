Sample Student Portfolio

There will be folder called SamplestudentPortfolio with 5 HTML files inside of it. When you double click this .zip file, it should automatically expand, creating a folder with 5 folders inside of it.

Go to documents Folder and create a folder called "[YourName]PortfolioSite". Copy Those 5 HTML files (not the folder into your new Site Folder on the computer) Go to Manage sites and click New Site For Site Name write: [Your Name] Portfolio. For the local site folder, choose the folder that you created.
